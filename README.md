# GPA Calculator (IUBAT)

A Python-based GPA calculator following the IUBAT grading system.

## Components & Weights
- Midterm: 25%
- Class Tests (CT): 10%
- Attendance: 5
- Presentation: 5
- Assignment: 5
- Final Exam: 50%

## Input Format
Provide the following values, each on a new line:
1. Midterm marks (out of 100)
2. CT marks (out of 100)
3. Attendance (out of 5)
4. Presentation (out of 5)
5. Assignment (out of 5)
6. Final marks (out of 100)

## Output
- Total Marks
- Letter Grade
- GPA

## Usage
```bash
python gpa_calculator.py
